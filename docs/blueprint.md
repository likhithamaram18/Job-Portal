# **App Name**: GradLink

## Core Features:

- User Authentication and Authorization: Secure user authentication with role-based access control (students, recruiters, admin) using Supabase Auth.
- Profile Management: Allow students to create detailed profiles with resume uploads and skill sets and recruiters to create company profiles.
- Job Posting: Enable recruiters to post job openings with detailed descriptions, locations, and job types, subject to admin approval.
- Job Search and Application: Provide students with advanced job search filters and the ability to apply to jobs, tracking application status.
- Admin Moderation: Implement an admin interface to approve/reject recruiter accounts and job posts, managing users and platform analytics.
- Supabase: Using Authentication, database, storage and RLS security using supabase backemd tool.

## Style Guidelines:

- Primary color: Dark blue (#2C3E50) to convey professionalism and trust.
- Background color: Light gray (#F4F4F8) for a clean, modern feel.
- Accent color: Teal (#3498DB) to highlight key actions and elements.
- Headline font: 'Space Grotesk', a proportional sans-serif font that conveys a techy and scientific feel.
- Body font: 'Inter', a grotesque-style sans-serif font.
- Use a consistent set of icons from a library like FontAwesome or Material Icons.
- A clean, responsive layout optimized for desktop and mobile devices.
- Subtle animations for transitions and user interactions to enhance engagement.
